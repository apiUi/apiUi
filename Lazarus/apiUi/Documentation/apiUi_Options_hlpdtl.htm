General
General options
Context

See also Menu->Project->Context properties
Use context properties to easily switch between endpoints, queuenames, filenames.
This option is also in use during opening of a project, useful when reading formal descriptions (wsdl's, api's) via http
StartAfterOpeningProject
Default 'true';
When 'true' apiUi will start running (activate itself) immediately after a project is opened
CreateBackup
Default 'true';
When 'true' apiUi will create a backup on saving an existing project (a backup is a folder that has an extra '~' at the end of its name)
ConfirmRemovals
Default 'true';
When 'false' apiUi will not prompt for certain removals
ConfirmTemporaryInactivity
Default 'true';
Default apiUi will prompt when certain actions require temporary inactivity.
When "false", apiUi will not prompt
ScrollExceptionsIntoView
Default 'false';
When 'true' apiUi will if necessary scroll to show the last created exception record in the exception log view and will show the Exception tab when an exception occurs.
CheckScriptAssignments
Default 'false';
When 'true', apiUi will check assignments from scripts agaist the Xsd and raise an exception when not all datatype restrictions are met.
When 'false', those violations will be ignored.
Note that syntax errors will wil cause apiUi exceptions regardless of this setting.
InitialCollapseHeaders
Default 'false';
When 'true', treeviews of soapmessages will initially have collapsed soap headers.
CollapseXmlNodeOnUncheck
Default "true";
When "true", apiUi will collapse xml nodes on uncheck by the user
ExpandXmlNodeOnCheck
Default "true";
When "true", apiUi will expand xml nodes on check by the user

Mq
IBM(C) Websphere MQ;
Using Mq requires MQ to be installed on the machine where apiUi is running.
Both Local Client and Local Server are supported.
Use
Piclist; Specifies use of MQ local server or MQ local client connection
LocalClient = Requires MQ Local Client to be installed on the machine running apiUi. MQ Local Client is software provided by IBM for free. To be able to use queues on a server, that server must allow local client connections
LocalServer = Requires MQ Local Server to be installed on the machine running apiUi
Undefined =
MaxWorkingThreads
Default "15"

Max number of MQ working threads that will be active simultaniously
TaCo
TaCo - Tandem Communication Protocol
Developed for Rabobank by Lars Jansen and Jan Bouwman

pingpong
Enable PingPong if you want to keep your TaCo connection healthywhen no other requests are send using this transport protocol.
New settings become active on (re)connecting to the TaCo server.
Enabled
Default 'true'
interval
Default "300000" which is 5 miutes in milliseconds.
Minimum value "30000" which is half a minute in milliseconds.
Colors
Customise colors
Xml
UnassignedValues
Specifies the background color for unassigned (unchecked) XML values
CorrelationValues
Specifies the background color for correlation values (applies to the grid only)
ExpectedValues
Specifies the background color for expected XML values.
For stub-operation this applies to request values.
For driver-operation this applies to response values.
